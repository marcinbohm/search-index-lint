## Before first pre-alpha release

- Confirm README demo commands work.
- Confirm `CHANGELOG.md` describes the release.
- Confirm release artifacts are not promised unless published.
- Confirm security reporting instructions are acceptable.
- Confirm fixtures and examples contain only synthetic data.
- Confirm `search-index-preflight.future.example.yaml` still says config loading is not implemented.
